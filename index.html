UntrackMe

Privacy-focused URL redirector and unshortener

About This Project

This is an updated fork of the original UntrackMe project.

Key Features

  • Redirect tracking domains to privacy alternatives
  • Unshorten URLs without loading tracking links
  • Remove tracking parameters from URLs
  • Open-source and transparent
  • No ads or tracking

Supported Services

YouTube → Invidious
Twitter → Nitter
Instagram → Bibliogram
Google Maps → OpenStreetMap
Reddit → Teddit/RedLib
TikTok → ProxiTok
Medium → Scribe
Wikipedia → Wikiless
